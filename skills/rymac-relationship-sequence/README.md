# rymac-relationship-sequence

> Writes the 10-email sequence that turns a fresh opt-in into a trained reader: E1 delivers the promised thing instantly, E2 introduces you, E3-E10 land one idea and one link each — and nobody gets pitched before email 9.

A drop-in [Claude Code](https://docs.claude.com/en/docs/claude-code) skill from the **[RyMac Skills](../../README.md)** pack.

## What it does

Writes and wires the post-capture sequence whose job is training, not selling: the reader learns that opening your email always pays and clicking your link is always worth it. You get the full E1-E10 copy in a locked format (signature opener, one idea, exactly one link, a P.S. that presells tomorrow), the proven cadence (1, 1, 1, 2, 2, 3, 2, 3, 4 day waits: daily while trust forms, stretching as it solidifies), entry-point handling (lead magnet vs booked call), and the wiring spec for any CRM or email platform: one entry form, attribution per capture surface, one automation owning the sequence.

## When it fires

- "nurture sequence" / "welcome sequence"
- "email sequence for new leads"
- "what happens after the opt-in?"
- wiring an exit pop, lead magnet, or calculator gate to email

(Or force it with `/rymac-relationship-sequence`.)

## Install

Copy this entire `rymac-relationship-sequence` folder into your Claude Code skills directory:

- **Global** (available in every project) — recommended:
  - macOS / Linux: `~/.claude/skills/rymac-relationship-sequence/`
  - Windows: `C:\Users\<you>\.claude\skills\rymac-relationship-sequence\`
- **One project only:** `<your-project>/.claude/skills/rymac-relationship-sequence/`

Restart Claude Code. Done.

## Example

> "My exit pop delivers a free teardown PDF. Write the sequence that starts when someone grabs it."

E1 delivers the PDF with the reassurance line, E2 introduces you, E3-E10 alternate gifts and content, and the soft ask waits until E9.
